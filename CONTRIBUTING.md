# Contributing to Context Lens

Thanks for your interest in contributing!

## How to Contribute

**Please open an issue first before submitting a pull request.**

1. **Open an issue** - Describe what you'd like to add or fix
2. **Discuss the approach** - Get feedback on how it fits into the project
3. **Get approval** - Once approved, start working on your contribution
4. **Submit a PR** - Reference the issue in your pull request

This ensures your time isn't wasted on changes that might not be accepted.

## Development Setup

```bash
# Clone and install
git clone https://github.com/cornelcroi/context-lens.git
cd context-lens
pip install -e .

# Run tests
pytest tests/
```

## Code Style

- Follow PEP 8
- Use Black for formatting: `black src/ tests/`
- Add type hints
- Write tests for new features

## Questions?

Open an issue or check existing issues first.
